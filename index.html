Badminton Video Analyzer

An advanced machine learning project for analyzing badminton gameplay

Project Overview

This project combines my passion for badminton with my drive to build machine learning tools that solve real-world problems. The goal is to allow users to upload match footage and receive valuable insights such as:

  • Most common shot patterns per player
  • Shot types with highest success rates
  • Detailed performance analytics

The project is currently in active development, with all data preprocessing, pose detection, and ML pipeline work being done personally. While the code is still in development and hasn't been pushed to GitHub, a private repository can be shared upon request.

VideoBadminton Dataset

A comprehensive dataset specifically designed for badminton action recognition, featuring:

  • High-quality badminton footage
  • Fine-grained action categorization
  • Expert-labeled shot types
  • High resolution frames (1280x720px)

Current Pipeline

  • Pose extraction using OpenPose/MediaPipe
  • Player identification logic (in progress)
  • Shot-type labeling (partially labeled subset)
  • Classifier training with temporal pose data

Challenges & Solutions

Dataset Challenges

The VideoBadminton Dataset presented several challenges:

  • Variable frame counts (1-100+ frames per clip. Not the best to train a model on, say, only 1 frame of data)
  • Only filmed from one angle (the end goal is to be able to analyze videos from different angles, to make it easy for any amateur badminton player to analyze their own gameplay)
  • Background is the same for all clips
Solutions Implemented
  • Standardized input by extracting exactly 6 frames per clip. Discard clips with less than 6 frames.
  • Reduced computational load using pose estimation to extract body keypoints
  • Developed auxiliary model for player identification based on shuttle trajectory
